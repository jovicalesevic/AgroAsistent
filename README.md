# AgroAsistent 1.0

Web aplikacija za vođenje digitalne knjige polja, praćenje parcela iz eAgrara i brzi pristup agro-informacijama.

## Tehnologije

MongoDB, Express, Node.js.

## Instalacija

1. Kloniraj repozitorijum i uđi u folder projekta.
2. Pokreni `npm install`.
3. Kopiraj `.env.example` u `.env` i podesi vrednosti (npr. `MONGO_URI`, `PORT`).
4. Pokreni `npm start`.

## Pokretanje

```bash
npm start
```

Aplikacija će biti dostupna na `http://localhost:5000` (ili na portu iz `.env`).
